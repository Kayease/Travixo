# Icons (SVG)

Shared SVG assets. Paths: `/icons/file.svg`, `/icons/globe.svg`, `/icons/next.svg`, `/icons/vercel.svg`, `/icons/window.svg`.

- **file.svg** – document/file icon
- **globe.svg** – globe icon (e.g. destination/location)
- **next.svg** – Next.js logo
- **vercel.svg** – Vercel logo
- **window.svg** – window icon

Use in code via `src="/icons/globe.svg"` etc. Add new icons here with lowercase, hyphenated names.
