# Travixo – Travel & Tour

A Next.js travel and tour site focused on safari and adventure experiences. Browse destinations, tours, stays, and manage bookings with a modern UI.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project structure

- **`app/`** – Routes, layouts, and page components (App Router).
- **`components/`** – Reusable UI: layout (Navbar, Footer), home sections, room and checkout components, shared UI.
- **`public/`** – Static assets (images, icons).

Built with Next.js, React, and Tailwind CSS.
